# Game-pingpong
Game pingpong adalah Game yang terinspirasi dari tenis meja. 
Dimainkan pada layar dua dimensi dengan tujuan memantulkan bola menggunakan racket agar tidak melewati batas racket sendiri.Game ini dimainkan oleh 2 pemain, setiap pemain harus menggerakkan racket keatas dan bawah untuk menghalau bola yang memantul dari dinding permainan dan racket pemain lawan .Jika bola berhasil melewati racket pemain1, pemain1 akan kalah sedangkan jika bola melewati racket pemain2, pemain2 akan kalah.
