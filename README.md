# Game-pingpong
Game pingpong adalah Game yang terinspirasi dari tenis meja Game dimainkan oleh dua orang
dimainkan pada layar dua dimensi dengan tujuan memantulkan bola menggunakan paddle agar tidak melewati batas paddle sendiri.
Dalam game ini, bola bergerak bolak-balik di layar dan akan memantul ketika mengenai paddle atau dinding permainan. Pemain mengendalikan paddle atasâ€“bawah untuk menghalau bola. Jika bola berhasil melewati paddle pemain1, pemain1 akan kalah sedangkan jika bola melewati paddle pemain2, pemain2 akan kalah.
