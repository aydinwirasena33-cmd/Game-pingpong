# Game-pingpong
Game pingpong adalah Game yang terinspirasi dari tenis 
dimainkan pada layar dua dimensi dengan tujuan memantulkan bola menggunakan paddle agar tidak melewati batas paddle sendiri.Game ini dimainkan oleh 2 pemain, setiap pemain harus menggerakkan paddle keatas dan bawah untuk menghalau bola yang memantul dari dinding permainan dan paddle pemain lawan .Jika bola berhasil melewati paddle pemain1, pemain1 akan kalah sedangkan jika bola melewati paddle pemain2, pemain2 akan kalah.
